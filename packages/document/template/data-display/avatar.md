## API
[[[api zh
shape: 头像的形状。
size: 头像的尺寸。
bordered: 头像是否拥有边框。
backgroundColor: 头像背景颜色。
borderColor: 头像边框颜色。
slots.default: 头像内容。
]]]
[[[api en
shape: The shape of the avatar.
size: The size of the avatar.
bordered: Whether the avatar has a border.
backgroundColor: The background color of the avatar.
borderColor: The border color of the avatar.
slots.default: The content of the avatar.
]]]
